== 9.16.1 GA（2025 年 3 月 06 日）

Cloud Volumes ONTAP 9.16.1 正式版现已可在 AWS、Azure 和 Google Cloud 中部署和升级。



== 升级说明

阅读这些说明以了解有关升级到此版本的更多信息。



=== 如何升级

Cloud Volumes ONTAP的升级必须通过控制台完成。您不应使用系统管理器或 CLI 升级Cloud Volumes ONTAP 。这样做会影响系统稳定性。

link:http://docs.netapp.com/us-en/bluexp-cloud-volumes-ontap/task-updating-ontap-cloud.html["了解主机通知您时如何进行升级"^]。



=== 支持的升级路径

您可以在 AWS、Azure 和 Google Cloud 中从 9.15.1 版本升级到Cloud Volumes ONTAP 9.16.1。控制台将提示您将符合条件的Cloud Volumes ONTAP系统升级到此版本。



=== 停机时间

* 单节点系统升级最多需要 25 分钟的时间才能使系统离线，在此期间 I/O 会中断。
* 升级 HA 对不会造成中断，并且 I/O 也不会中断。在此无中断升级过程中，每个节点都会同时升级，以继续为客户端提供 I/O 服务。
